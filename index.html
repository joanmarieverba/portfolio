About Me

Hello, I'm Joan Marie Verba. I'm a Front End developer, and I believe that the goal of web development is to make applications/web pages that present users with the information they want in the style they prefer. I find this both challenging and exciting, and is one of the reasons I find coding so much fun.

Education

Graduate of the Iron Yard Minneapolis
The Iron Yard offers immersive, 12-week courses that go far beyond tactical skills and teach students to think as a software engineer. The process shows students to learn how to learn so as to keep current on technologies in the long run. The Front End Engineering course focuses on creating experiences in web browsers that users see and interact with. The lessons cover web design principles to complex JavaScript applications (including ReactJS) that manage data, in addition to HTML and CSS.
